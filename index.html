The GDP of Select International Cities

Source: Data from the Organisation for Economic Co-Operation and Development (OECD).

What is GDP?: GDP stands for Gross Domestic Product. This is defined as “the total value of goods produced and services provided in a country during one year.” However, this particular dataset focuses on selected cities and what their GDP would be.

Directions: To use the application, use the slider to change the year. You can click on the points to view additional information for the point. If you wish to change the basemap, this can be done using the icon in the top right corner of the map.
